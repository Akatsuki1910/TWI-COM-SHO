# TWI-COM-SHO
## Twitter Comment Show App
Twitterで指定したタグのtweetをニコニコ形式で表示するアプリです

## 使い方
タグを入力して左のボタンを押すだけ  
終わる時はインジケータから終了を押す  
この画面から終わるときはバツを押す

Windows
